# CSS3 Transitions, Animations, and JavaScript Functions

## Project Overview

This project demonstrates the use of **CSS3 transitions and animations** combined with **JavaScript functions** to create interactive and visually appealing web experiences. It is part of the PLP Academy curriculum for February 2025, focusing on enhancing front-end development skills.

## Features

- **CSS3 Transitions**: Smooth state changes for elements such as hover effects and color transitions.
- **CSS3 Animations**: Keyframe-based animations for dynamic visual effects.
- **JavaScript Functions**: Interactive behaviors triggered by user actions, such as button clicks or page events.

## Project Structure

```
/web/feb-2025-css3-transitions-animations-js-functions-Collins101-dev/
├── index.html       # Main HTML file
├── styles.css       # CSS file for transitions and animations
├── script.js        # JavaScript file for interactive functions
└── README.md        # Project documentation
```

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/PLP-WebTechnologies/feb-2025-css3-transitions-animations-js-functions-Collins101-dev.git
    ```
2. Navigate to the project directory:
    ```bash
    cd feb-2025-css3-transitions-animations-js-functions-Collins101-dev
    ```
3. Open `index.html` in your browser to view the project.
